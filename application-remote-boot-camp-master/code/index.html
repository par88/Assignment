BROWSER

A browser is an application software that allows a computer user to access and interact with the information on the World Wide Web, including web pages, videos, images and other files. When a user wants to access a particular web page, the web browser that runs on the user’s device, contacts the Web server and requests information. The Web server sends the information back to the Web browser which interprets and displays the results on the user’s device.

The most used browser at the moment is Google Chrome followed by Safari. Some other browsers include:

  • Firefox
  • Microsoft Edge
  • Internet Explorer
  • Lynx
  • Opera
  • Flock
